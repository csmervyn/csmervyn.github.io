如何修改最后一次 commit 的 author 信息

January 20, 2024 阅读时间：1 分钟 | 🍉 32 字
mervyn
git

在新 repository 提交代码时，有时忘记配置该 repository scope 内的 user.name 和 user.email，从而导致 commit 的 author 信息为 global 配置的 user.name 和 user.email。如果此时想要修改最新提交的 author 信息，可以使用下面的 command:

git commit --amend --author="{username} <{email}>"

例如：

git commit --amend --author="csmervyn <csmervyn@gmail.com>"
