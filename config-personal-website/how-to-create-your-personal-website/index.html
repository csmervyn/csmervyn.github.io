创建博客 Link to heading

hugo new site csmervyn-blog

在当前目录初始化一个空的 git 仓库。

cd csmervyn-blog
git init

配置 Hugo 的主题 Link to heading

选择主题：Hugo 主题

将主题应用与自己的博客，在我们的示例中我们使用 hugo-coder 这个主题。

git submodule add https://github.com/luizdepra/hugo-coder.git themes/hugo-coder

在配置文件中表明自己的主题。

echo "theme = 'ananke'" >> config.toml
